Sales Performance Dashboard Project

📍 Project Title

Sales Performance Dashboard – Excel \& Power BI



📘 Project Overview

This project delivers a complete end‑to‑end analysis of business performance using Excel and Power BI.

The workflow begins with data cleaning and exploration in Excel, followed by building an initial analytical dashboard using PivotTables.



The cleaned dataset is then imported into Power BI to create three interactive dashboards that analyze sales, profitability, and geographic performance.







🧹 1. Excel Data Cleaning \& Preparation

The project started in Excel, where the raw dataset was cleaned and prepared for analysis.



✔ Key Cleaning Steps

* Removed duplicates
* Standardized category, sub‑category, state, and city names
* Fixed inconsistent formatting
* Converted data types (Sales, Profit, Quantity)
* Handled missing values
* Created calculated fields (Total Sales, Total Profit, Total Quantity)



✔ Excel Data Modeling

* Structured tables
* Added named ranges
* Ensured clean, analysis‑ready data for Power BI import







📊 2. Excel Dashboard (PivotTables)

Before moving to Power BI, an analytical dashboard was created in Excel using PivotTables.



✔ PivotTables Included

* Sales by Category
* Profit by Category
* Sales by Region
* Top Sub‑Categories
* Discount impact analysis



✔ Purpose

To validate the dataset, explore initial insights, and ensure the data structure is ready for Power BI.







🔄 3. Importing Data into Power BI

The cleaned Excel table was imported into Power BI as:



cleaned\_data

Additional transformations were applied using Power Query:

* Verified data types
* Ensured geographic fields were clean
* Prepared the dataset for mapping visuals







📊 4. Power BI Dashboards

The project includes three fully interactive dashboards:



1️⃣ Sales Performance Dashboard (Power BI)

Provides an overview of sales performance across product categories, sub‑categories, and regions.



Key Visuals:

* Total Sales KPI
* Sales by Category
* Sales by Sub‑Category
* Sales by Region
* Top 10 Products by Sales



Purpose:  

Identify which categories, sub‑categories, and regions generate the highest sales.





2️⃣ Profitability Analysis Dashboard

Analyzes profit, discount impact, and margin performance.



Key Visuals:

* Total Profit KPI
* Avg Profit Margin
* Avg Discount
* Profit by Category
* Discount vs Profit
* Top 10 Most Profitable Sub‑Categories
* Profit Margin by Category



Purpose:  

Understand profitability drivers and how discounts affect profit.





3️⃣ Geographic Sales \& Profit Analysis Dashboard

Provides a geographic breakdown of sales and profit across states, regions, and cities.



Key Visuals:

* Sales by State (Map)
* Profit by State (Shape Map)
* Sales by Region
* Profit by Region
* Top Cities by Sales
* Profit by Region and Category (Matrix)



Purpose:  

Identify top‑performing locations, regional trends, and geographic opportunities.

🎯 Key Insights

* Technology category generates the highest profit.
* Office Supplies show strong profit margins.
* Higher discounts reduce profitability.
* West region leads in both sales and profit.
* Certain states and cities outperform others significantly.





🚀 How to Use the Dashboard

* Open the .pbix file in Power BI Desktop.
* Navigate between the three dashboard pages.
* Use slicers and filters to explore specific segments or regions.
* Hover over visuals to view detailed tooltips.

This project demonstrates a full analytical workflow starting from Excel and ending with a professional multi‑page Power BI dashboard.
